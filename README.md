#Music Player Project 🎵
A sleek and interactive Music Player application built with HTML, CSS, and JavaScript. This project was developed as part of an internship task to demonstrate frontend development skills and audio handling in web browsers.

##🚀 Features
Custom Playlist: Easily select and play tracks from a built-in playlist.

Interactive Controls: Smooth Play/Pause functionality with real-time feedback.

Responsive Design: A modern, glassmorphism-inspired UI that looks great on all devices.

Dynamic Content: Seamless switching between songs with updated track information.

##🛠️ Technologies Used
HTML5: Structuring the player interface.

CSS3: Styling, layout, and visual effects (gradients and border-radius).

JavaScript: Managing audio playback, playlist logic, and event handling.
